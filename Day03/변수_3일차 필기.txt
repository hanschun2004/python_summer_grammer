상수
- 데이터를 변경할 수 없는 공간
- 리터럴 상수, 심볼릭 상수 X

리터럴 상수
- 데이터 그자체
- 일정한 형식의 공간 자동으로 만들어지고 데이터가 자동으로 저장된다. 
- 공간을 구분할 수있는 방법이 없다.
- 공간을 다시 이용할 수있는 방법이 없다. 

변수- 지속성 데이터
- 데이터 하나를 저장하는 공간
- 직접 일정한 형식의 공간을 만들고 우리가 직접 저장시켜주어야 한다. 
- 공간의 이름을 통해서 다시 이용할 수가 있다. 

선언 
- 일정한 형식의 변수공간을 할당받고 그 변수 공간에 변수명을 붙이는 작업
- 반드시 선언하면서 초기화를 진행하여야 한다. 
- 선언 형식
	변수명 = 값  ex) name = "전종욱"
	- 변수명 : 변수 공간을 대표하는 이름.. 변수 호출할때 사용
	- 데이터의 성격이 연상되는 단어로 이름을 지어준다.
	- 소문자를 권장한다.
# 작명 규칙
- 알파벳, 숫자, _(언더바)로 구성된다.
- 숫자로 시작하면 안된다.
- 대소문자 구별 가능
- 공백이나 특수기호는 사용 할 수 없다. *** ex) my Name X,  my_Name O
- 예약어는 이름으로 사용할 수 없다. ex) for, if, while 

# = (대입 연산자) : 우측공간의 값을 좌측 공간에 대입하는 연산자 
- 초기화 
 - 할당받은 변수공간에 값을 처음 넣는 작업
 - 할당받은 변수공간은 항상 쓰레기값으로 채워진다.
 - 공간의 데이터를 이용하려 할 때 에러발생(파이썬은 무조건 선언과 동시에 초기화가 진행되어야 한다.)

- 호출
 - 할당받은 변수공간을 불러오는 작업
 - 형식
         변수명 

# 자료형 ( Data Type )
- 컴퓨터는 정수 박에 처리 하지 못한다.
- 데이터를 저장하는 방식이 데이터마다 다르기 때문에 각각의 데이터에 맞는 공간들이 존재한다. 
- 공간을 만들기 위한 틀
- 데이터의 형태를 구분하고 데이터의 범위나 부호의 사용여부등을 결정하는것 
- 공간의 형식을 정의하는 것
- 파이썬은 자기가 알아서 공간을 만들어준다. ***

- 파이썬에서의 자료형의 종류
 - bool(논리형) : 참(True)과 거짓(false)을 저장하고 관리하는 자료형
 - int(정수형) :  정수를 저장하고 관리하는 자료형 
 - float(실수형) :  실수를 저장하고 관리하는 자료형
 - str(문자열) :  문자열을 저장하고 관리하는 자료형
 - list(데이터의 집합) :  데이터의 변경 가능 
 - tuple(데이터의 집합): 데이터의 변경 불가능
 - dictionary(데이터의 집합) : key와 value  값이 쌍으로 이루어 관리
